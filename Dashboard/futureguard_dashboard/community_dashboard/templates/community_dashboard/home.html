Follow the public workflow

The platform is organized into clear steps so a normal community user knows exactly what to do next.

  • Start at Home

    Read the introduction so you understand the platform's purpose and limits.

  • Go to Predictions

    Enter a climatic region in the top search bar, then review the prediction output and response guidance.

  • Use Emergency Report for real incidents

    Submit emergencies when something urgent is actively happening on the ground.
